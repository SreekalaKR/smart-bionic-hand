
# 🦾 Smart Bionic Hand – RNN Based Prosthetic Control System

## 📌 Project Overview

As my main project during my B.Tech journey (2023), I developed a **Smart Bionic Hand system** aimed at improving prosthetic hand control using deep learning techniques.

The system integrates **Electromyography (EMG) signals** and **flex sensors** embedded in a data glove to capture hand movements and improve motion recognition accuracy.

A **Recurrent Neural Network (RNN)** model is used to process sequential sensor data, enabling more precise and adaptive gesture prediction.

This project combines **technology and human-centric design**, focusing on improving accessibility and quality of life for individuals requiring prosthetic support.

---

## ⚙️ Key Features

- EMG signal-based motion detection  
- Flex sensor integration using data glove  
- Real-time gesture recognition  
- RNN-based deep learning model for sequence prediction  
- Improved accuracy in motion classification
